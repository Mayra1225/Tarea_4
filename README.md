# 🎓 Sistema de Gestión de Notas - MVC + Vue.js + .NET Core

Este proyecto es una aplicación web desarrollada para la gestión de alumnos y cálculo de notas académicas según ponderaciones establecidas:

- P1 = 25%
- P2 = 35%
- P3 = 40%

El sistema permite registrar alumnos, calcular la nota necesaria en el tercer progreso para aprobar con mínimo 6, y visualizar reportes generales.

---

# 🚀 Tecnologías utilizadas

## Backend
- ASP.NET Core Web API
- C#
- Arquitectura MVC
- Servicios en memoria (sin base de datos)
- CORS habilitado

## Frontend
- Vue.js 3
- Axios
- HTML5 + CSS3
- Diseño tipo dashboard

## Deploy
- Backend: Render
- Frontend: Netlify
- Repositorio: GitHub (monorepo)
